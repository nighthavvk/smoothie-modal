Get started

Add CSS and JS files to your document (don't forget about jQuery) 

<link rel="stylesheet" href="css/smoothie-modal.css">

<script src="js/smoothie-modal.js"></script>

You can also use our default theme 

<link rel="stylesheet" href="css/smoothie-modal-theme.css">

Wrap your content using js class helper - js-modal-wrapper 

<div class="js-modal-wrapper"></div>

Follow next stucture to get basic functionality. Use data attribute
(data-modal) to manage your modal windows.

Use class js-modal-open for element to open modal window on click and js-modal-close to close it. 

<button class="js-modal-open" data-modal="default">Demo</button>

<div class="smoothie-modal js-modal-window" data-modal="default">
  Your content here...
</div>
